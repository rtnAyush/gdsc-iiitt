## Getting Started
1. Folk the repo.
2. Clone your floked repo on the local machine.
   ```git clone <github url>```
3. Install depedenies using ```npm install .```
4. Run the dev mode as ```npm run dev```
5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
6. You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
